jwt (Json Web Token)

2017-05-22
Go Golang Web

最近做網頁需要用到token來做身份認證，之前都是用cookie，但感覺不太安全，因此找到了jwt這個有趣的token

什麼是jwt?

jwt是一種以json格式加密而成的token形式，底下先來個範例

從上面的範例可以發現jwt用.分成三段

  • 第一段是加密的形式
    • 用base64加密
  • 第二段是自帶的訊息
    • 用base64加密
  • 第三段是驗證用的token
    • 以第一段所述的加密方式來加密

第一段

透過base64解密後

  • alg代表的是加密演算法
    • HS256代表HMAC + SHA256
  • typ表示token的形式

第二段

透過base64解密後

  • exp代表過期的時間
  • iss是簽token的人

其餘的資訊是在建立token時一起放進去的其他訊息，用途是給要求token的人看的，可以放入任意的訊息

第三段

第三段是用第一段所說的 HMAC + SHA256 將前面兩段加密而成

驗證方法？

驗證的時候就是再做一次加密，如果出來的token跟原本的一樣的話，那麼就當作這個token是可用的，接著再檢查是否過期

被盜用？

加密的時候要使用一個private key，若使用不同private key加密的話就會得到不一樣的token，另一方面，如果改動第二段所存的資料，那麼加密出來的token就會與原先的不同，因此可以保證token的可用性

換句話說，如果對方拿到private key的話，就可以冒用自己的名義去簽發token喔
